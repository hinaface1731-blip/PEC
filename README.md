# ПЭК — Полярная Экспедиционная Компания

Корпоративный сайт геологоразведочной компании полного цикла. Геология, геофизика, бурение, маркшейдерия, лаборатория, проектирование, экология. 17 лет работы в Арктике и Сибири.

**Демо:** [https://polar-ec.ru]

## Стек

- **Next.js 16** — App Router, серверные компоненты, API Routes
- **TypeScript** — строгая типизация
- **Tailwind CSS 4** — стилизация
- **shadcn/ui** — UI-компоненты на базе Radix UI
- **Framer Motion** — анимации
- **Яндекс Карты 2.1** — интерактивная карта географии работ
- **Vercel Analytics** — аналитика
